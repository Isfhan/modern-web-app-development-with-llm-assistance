DNS (Domain Name System)
The Domain Name System (DNS) is like a phone book for the internet. It converts website names into IP addresses.
HTTP/HTTPS
HTTP (Hypertext Transfer Protocol) is how web browsers and servers communicate. HTTPS adds security through encryption.
Client-Server Architecture
The internet works on a client-server model. Your device (client) requests information from servers that host websites and services.
Web Browsers
Web browsers are programs that let you access websites. They use different engines to display web content.
